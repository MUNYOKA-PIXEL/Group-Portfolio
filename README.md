# Group Portfolio

A public portfolio website for GroupAlpha, built with plain HTML, CSS, and JavaScript.

## Live site
🌐 **[View Live](https://munyoka-pixel.github.io/Group-Portfolio/)**

This project is published with GitHub Pages.

## Features
- Single-page application with smooth view switching
- Responsive design optimized for mobile and desktop
- Contact form with validation and Formspree integration
- Animated reveals and rotating hero messages
- Modern dark theme with gradients and glow effects
- Toast notifications for form feedback
- Bright button state indicating form readiness

## Technologies Used
- HTML5
- CSS3 (with custom properties, Grid, and Flexbox)
- Vanilla JavaScript (ES6+)
- Formspree for form submissions

## Setup and Installation
1. Clone the repository: `git clone https://github.com/munyoka-pixel/Group-Portfolio.git`
2. Open `index.html` in your browser to view locally.
3. Live site is automatically deployed to GitHub Pages.

## Customization
- Update team photos in the `about` section (replace `Mary.jpeg` and `Nicolus.jpeg`).
- Modify project details in `index.html` to reflect your actual work.
- Adjust colors and styles in `style.css` using CSS custom properties.
- Update contact information and form endpoint in the contact section.
